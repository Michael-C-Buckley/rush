rush

Rush is Bash with better UX.

Use the shell language you already know. Spend less time fighting the prompt.

$ rush
# bash-compatible syntax, friendlier interface

Start here

Current focus:

  • Bash compatibility
  • better interactive UX
  • POSIX shell behavior

Install

git clone https://github.com/rockorager/rush
cd rush
zig build -Doptimize=ReleaseSafe --prefix $HOME/.local

Docs

  • Advanced features: adaptive themes, terminal protocols, and interactive UX.

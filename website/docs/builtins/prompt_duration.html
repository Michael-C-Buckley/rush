PROMPT_DURATION(1)

NAME

prompt_duration - print the previous foreground command duration for prompt rendering.

SYNOPSIS

prompt_duration

DESCRIPTION

prompt_duration prints the previous foreground command duration in milliseconds, such as 123ms. It is only available while Rush is rendering the interactive prompt.

EXIT STATUS

Zero when a duration is available, one when no previous command duration is available, and two for usage errors.

SEE ALSO

rush-builtins(7), prompt(1), prompt_pwd(1), rush configuration.
